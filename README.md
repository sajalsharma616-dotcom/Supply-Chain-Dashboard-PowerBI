# Supply-Chain-Dashboard-PowerBI
An interactive Power BI dashboard tracking supply chain inventory, sales trends and product performance.

## Overview
An interactive Power BI dashboard built to track and analyse 
supply chain inventory and sales performance across multiple 
product categories.

## Dashboard Features
- KPI cards showing total inbound quantity, outbound quantity 
  and net stock levels
- Bar chart comparing net quantity by product name and category
- Line chart showing monthly sales trends (April–May)
- Bar chart of units sold by product name
- In vs Out quantity comparison chart
- Interactive category filter/slicer

## Data Model
The dashboard uses 3 connected tables:

| Table | Description |

| Inventory Management | Daily stock inflow and outflow by product |
| Product List | Master data — product name, category, supplier, unit price |
| Sales Data | Monthly units sold by product |

## Tools Used
- Microsoft Power BI Desktop
- Microsoft Excel (data source)
